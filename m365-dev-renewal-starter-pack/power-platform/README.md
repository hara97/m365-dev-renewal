# Power Platform (Power Apps + Power Automate)

This folder documents Power Platform activity in the dev tenant.

### Steps:
1. Go to [Power Apps](https://make.powerapps.com) → create a **Canvas app from blank**.  
2. Connect to sample data (Excel, SharePoint, etc.).  
3. Save & publish.  
4. Create a Power Automate flow:
   - Trigger: When a file is created in OneDrive
   - Action: Send message in Teams
5. Export the flow definition as JSON and save it here.

### Files
- `flow-template.json` – example exported flow definition
