# Office Add-in (Excel Demo)

This folder contains a simple Office Add-in manifest for Excel Online.

### Steps
1. Install the Office Yeoman generator:
   ```bash
   npm install -g yo generator-office
   yo office
   ```
2. Choose **Excel → Taskpane → JavaScript**.
3. Test by sideloading manifest.xml into Excel Online.

### Files
- `manifest.xml` – example add-in manifest
- `screenshots/` – add screenshots of the task pane in Excel
