# Microsoft Teams Demo App

A simple personal Teams app sideloaded in the developer tenant.

### How to use
1. Open Microsoft Teams → **Apps** → **Developer Portal**.  
2. Create a new app named *m365-renewal-demo*.  
3. Add a **Personal tab** → link to https://www.bing.com (for testing).  
4. Export the manifest and upload it here.

### Files
- `manifest.json` – Teams app manifest (from Developer Portal).  
- `screenshots/` – add screenshots of the app in Teams.
