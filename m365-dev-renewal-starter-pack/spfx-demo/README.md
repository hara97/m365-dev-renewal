# SharePoint Framework (SPFx) Demo

This folder contains the SharePoint Framework Hello World web part.

### Steps
1. Open PowerShell or terminal.
2. Run:
   ```bash
   yo @microsoft/sharepoint
   gulp serve --nobrowser
   ```
3. Open `https://localhost:4321/temp/workbench.html` and test.
4. Deploy the web part to your SharePoint app catalog.

Add screenshots of your web part in the `screenshots/` folder.
